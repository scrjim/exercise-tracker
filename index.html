setup
One-time setup required:

To activate the coach you need a free Anthropic API key:

1. Go to console.anthropic.com
2. Sign up or sign in
3. Click API Keys in the left menu
4. Click Create Key and copy it
5. Paste it in the field below and click Save

Your key is saved locally in your browser and never shared with anyone. You only need to do this once per device.
